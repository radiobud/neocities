This is my section for influences: sound effects, tutorials, and inspirations.

I first got inspired to make this site after watching the "Be your own algorithm" video essay by youtuber pagemelt. Neocities was one of the sites mentioned in the video.

The first Neocities site I visited was Blamensir's Monastery, and it was so magical - I wanted to create a website too, even if much more mundane.

There seem to be so many flourishing Neocities sites which share, as one of their goals, inspiring and teaching others to try the same. So many sites have tutorials or resource pages which pass on inspiration and tips. I'm grateful for the work so many folks have done to give a leg up.

I haven't tried to learn CSS, or really expand past links and images in html, since the days of trying to make a page on Neopets to host all my screenshots. So this is both a nostalgia trip and an accomplishment: HTML, CSS, and now even Javascript have been demystified for me, and I'm learning to use them comfortabley. Even if there seem to be a lot more paradigms to wrangle in order to make the dern thing work on multiple screen sizes.

Onward!
